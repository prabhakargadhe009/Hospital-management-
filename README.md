# 🏥 Hospital Management System – CityCare

A modern and responsive **Hospital Management System** built using HTML, CSS, and JavaScript. This application allows users to book appointments, manage records, generate receipts, and interact with an AI assistant.

---

## 🚀 Features

- 📅 Book appointments بسهولة
- 🧾 Generate & print receipts (PDF supported)
- 🏥 Multiple departments with consultation fees
- 📊 View all appointments in a table
- ❌ Delete / cancel appointments
- 🤖 Built-in AI assistant for help
- 💾 Data stored using browser LocalStorage
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage (for saving data)
